# Modern Navbar Collection

A curated collection of 15-20 modern, fully responsive, and production-ready navigation bars. Built with clean HTML, CSS, and JavaScript, these navbars are designed to be easily copied and integrated into your frontend projects.

[![Deploy Showcase Site](https://github.com/USER/modern-navbar-collection/actions/workflows/deploy.yml/badge.svg)](https://github.com/USER/modern-navbar-collection/actions/workflows/deploy.yml)

## Features
- **Production-Ready**: Accessible, responsive, and cross-browser compatible.
- **Mobile-First**: Designed for optimal viewing on mobile devices with smooth transitions to desktop layouts.
- **Copy-Paste Simplicity**: No complex frameworks required. Pure HTML, CSS, and minimal Vanilla JS for behavior.
- **Variety of Styles**: Includes Glassmorphism, Mega Menus, Sticky styling, Sidebar navs, and more.

## Installation / Usage

1. Browse the [Live Demo](#) to find a style you like.
2. Click the "Live Preview" button on the navbar card.
3. Review the code components (HTML, CSS, JS).
4. Click "Copy" and paste the code directly into your project.
5. Some navbars utilize [Ionicons](https://ionic.io/ionicons) for icons, and Google Fonts for typography. Make sure to include the provided `<script>` and `<link>` tags in your `head` if you want the exact same look.

Alternatively, you can clone the repository to view all the raw files:
```bash
git clone https://github.com/USER/modern-navbar-collection.git
```
The raw vanilla HTML/CSS/JS code is located in the `navbars/` directory.

## Local Development (Showcase Site)

If you wish to run the showcase React site locally:
1. Navigate to the `showcase-site` folder.
2. Run `npm install`
3. Run `npm run dev`

## Contributing
Contributions are welcome! If you have a cool navbar design, feel free to open a Pull Request.
1. Fork the repo.
2. Create a folder in `navbars/` with your distinct name.
3. Add `index.html`, `style.css`, and optionally `script.js`.
4. Register your new navbar in `showcase-site/src/data.ts`.
5. Ensure it's fully responsive and accessible.
6. Submit a PR.
